# Spoken Hook Types

Secret Reveal (Educational) Starting Point: Something new happened that you want to breakdown or explain

Case Study (Educational) Starting Point: Some person/company accomplished a relevant dream outcome using a certain method

Problem (Educational) Starting Point: Some problem that the viewer has and will want solved

Contrarian (Educational) Starting Point: Some non-obvious take you have that contradicts the mainstream viewpoint

Negative (Educational) Starting Point: Some conditional action that you know the viewer should take before continuing the common path

Education (Educational) Starting Point: Some educational outcome that you want to explain through the content

List (Educational) Starting Point: Some list of options that will help the viewer accomplish some outcome

Scenario / Hypothetical (Educational) Starting Point: Some hypothetical scenario that allows you to paint an ideal outcome/solution

Comparison (Educational) Starting Point: Some comparison of two options (one nonobvious one that is working vs one more obvious that isn’t

Question (Educational) Starting Point: Some question the viewer is pondering that is blocking them from achieving a core outcome

Ranking/Rating (Educational) Starting Point: Some set of topics or items that need to be ranked/rated

Authority (Storytelling) Starting Point: Starting with some core desired outcome (from creator 1st person POV)

Personal Experience (Storytelling) Starting Point: Starting with some personal problem or experience
