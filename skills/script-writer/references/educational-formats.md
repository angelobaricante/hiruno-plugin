# Educational Formats

Breakdowns/Explainers (Educational)

- these videos start with a topic/concept/event you want to explain

Case Studies (Educational)

- these videos start with an example of success you want to breakdown

Problem Solution (Educational)

- these Videos start with a problem you want to solve

Common Trap/Mistake (Educational)

- these videos start with a common trap or mistake you want to help the viewer avoid

Tutorial (Educational)

- these videos start with a skill or process you want to break down in steps

Listicle (Educational)

- these videos start with a list of options for accomplishing some task/problem

Scenarios (Educational)

- these videos start with a hypothetical scenario you want to explore

A vs B (vs C) Comparison (Educational)

- these videos start with a comparison of two or three things

Q & A (Educational)

- These videos start with a question you’re going to answer

Interview Clips (Educational)

- These videos start with a question from the interviewer

Ranking/Rating/Tier List (Educational)

- These videos start with a list of options you’re comparing to each other

Levels (Educational)

- These videos start with a set of options presented in ascending order

Reaction (Educational)

- These videos start with a base video you’re reacting to
