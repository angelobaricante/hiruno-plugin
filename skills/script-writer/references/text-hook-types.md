# Text Hook Types

Stated Benefit/Outcome (Substance) Starting Point: This is title text that emphasizes the outcome/benefit from the core video subject

Video Subject (Substance)  
Starting Point: This is the title text that reiterates the subject of the video

Core Takeaway (Substance)  
Starting Point: This is title text that emphasizes the core takeaway from the video

Core Question (Substance)  
Starting Point: This is the title text that emphasizes the core question that will be answered in the video

Series Name (Substance)  
Starting Point: This is the title text that labels the name of a Series

A vs B Comparison Labels (Substance)  
Starting Point: These are title text that help label the A vs B comparison scenario

Core Topic (Single Font, Solid Background (Layout/Motion) Starting Point: This is title text that states the core topic, desired in plain font with a color background

Core Message (Dual Font) (Layout/Motion)  
Starting Point: This is title text that reiterates the core message of the video, designed in a dual font at the top of the screen

Core Takeaway (Single Font, Aesthetic) (Layout/Motion)  
Starting Point: This is title text that states the core takeaway of the video, designed in a single aesthetic font

Words Added Via Music (Dual Font) (Layout/Motion) Starting Point: This is title text that is added to the video as the music hits the beat

Sliding In (Layout/Motion) Starting Point: This is title text that slides onto the screen

A vs B Fixed (Layout/Motion)  
Starting Point: These are title text that help label the A vs B comparison scenario that are fixed on screen beneath each side of the comparison

Tracking Captions (Word for Word) (Layout/Motion)  
Starting Point: These are captions that appear integrated with the visuals and move as the visual hook moves

First Few Locked, Next Few 1 by 1 (Layout/Motion)  
Starting Point: This is when the first couple words of the text hook start out on the screen, but the next few are added as they’re said

Full Sentence Show, Single Word Emphasis (Layout/Motion)  
Starting Point: This is when the spoken hook is displayed all at once in fixed text, but one word is emphasized or larger than the rest

Full Sentence Show, Plain Formatting (Layout/Motion)  
Starting Point: This is conventional Tiktok title text font that is locked to top of the screen
